    Stable

    Change Display Brightness

     Changes brightness of primary display based on the time of day, using values that can be customized in the script.     		@t1m0thyj Download
    Requires hardware that supports DDC/CI (the ability to change display brightness in Windows).

    Change Windows App Theme

     Changes Windows 10 app theme (used by Store apps) based on the time of day.     		Windows 10 Desktop Only @t1m0thyj Download

    Change Windows System Theme

     Changes Windows 10 system theme (used by taskbar and Start menu) based on the time of day.     		Windows 10 Desktop Only @t1m0thyj Download

    Experimental

    Change Lockscreen Image

     Changes lockscreen image dynamically to match desktop wallpaper.     		Windows 10 @t1m0thyj Download
    Changing the image does not work consistently. (MSDN)
    Requires that the "Background" option is set to "Picture" in the Windows 10 lockscreen settings.

    Change Microsoft Edge Theme

     Changes Microsoft Edge theme based on the time of day.     		Windows 10 Desktop Only @t1m0thyj Download
    Changing the theme does not update it in Edge immediately, restarting the browser is required for the change to take effect.
